Overview

This website describes the activities conducted for each of the phases of this research.

Study area

The study area was defined as the intersection of Colombia international boundary and the Llanos savannas ecoregion (Olson and Dinerstein 1998, Olson et al., 2001) identified by World Wildlife Fund (WWF).

Phase 1: Spatial dynamics and drivers of land-cover change

We've crafted some handsome templates for you to use. Please see the description of the data sources, methods and results for this phase.

Phase 2: Impacts of land-cover change

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. Please see the description of the data sources, methods and results for this phase.

Phase 3: Future scenarios of land-cover change and corresponding impacts

You can @amsantac a GitHub username. Please see the description of the data sources, methods and results for this phase.
